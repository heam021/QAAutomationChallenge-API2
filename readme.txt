
Project: QAAutomation Challenge API 2 
Playwright.

This project requires the following:
Install Visual Studio Code.
Install Node 

Inside VS
First, make sure you have a package.json in your project. If you don't have it, you can create it by running "npm init" in the root of your project.

You need to have Mocha installed with the command:
 npm install -g mocha chai

You need to have Axios installed with the command:
npm install axios

To be able to run the project you need to use the command 
npc mocha.
